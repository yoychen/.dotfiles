# Yoychen's Dotfiles

It is used to personalize my system.

- utils
  - nvm
  - pyenv
  - docker
  - docker-compose
  - zsh (with oh-my-zsh)
- config
  - zshrc
  - gitconfig
  - vimrc

## Setup

1. Clone this repo into your home directory

2. Run
    ```bash
    cd ~/.dotfiles
    ./setup.sh
    ```
